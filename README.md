# kombu

To get started, run the following:
```bash
cd kombu
make venv
source venv/bin/activate
export OPENAI_API_KEY=[ADD_KEY_HERE]
```

To generate sample_sessions:
```bash
python generate_sample_session_data.py
```

To run Streamlit app:
```bash
make run-streamlit
```

To deactivate venv:
```bash
deactivate
```
